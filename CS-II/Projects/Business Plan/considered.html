Considered Proposals

Extracurricular Student Interaction App

Extracurricular Student Interaction App was the old idea of Even Bowerman before he joined us. It's a website where students can obtain information about school related activites. It would be a cool idea, but there are a lot of websites that are similar to what he proposed.

Available Tables

This project is an app that books online reservations at resturants without the need to call the resturant. I considered this project because I too, dislike calling up resturants to make reservations. I did not join however, due to previous plans to work on another project.

Code Converter

This project is a code converter. It will take in code and convert it into other programming languages. This project seems very challenging and will require a lot of knowledge and research of numerous programming languages. I considered this one becuase of the challenging aspect.

Pair Search Board

Making a brain game seems interesting enough. It's also challenging without being to much of a stretch. I assume it would be 2D, so it may be easier than making a 3D game.

Unity Game

I heard that Unity is a fun engine to learn, and many indie devs use it, but since the proposer didn't specify on what type of game he is planning to make, I wouldn't be joining it.

Souls-like Game

I'm not very experience in video games, though I have watched gameplays. I think a game similar to Dark Souls would be fun to make, but depending on the size of the map, it would need lots of work.

Foob

I would like to join in on the project because I'm good friends with the person who proposed the idea, but he already has two other group members, so I don't think he would want an extra person.

ShelterSafe

The projects seems interesting because natural disasters were always fascinating to me. However, I already have plans to work on a game with someone else.

3D Rendering System

I think this projects is quite ambitious, but it could definitely be done since the proposer knows his stuff. Sometimes, devices have trouble loading whenever playing a 3D game. However, I have other plans.

Roguelike Game

I have decided on this proposal because it would be challenging, but can be done within the reasonable time limit. I haven't played a roguelike game myself, but watched gameplay videos and it's quite action-packed, which I like. I still don't agree with adding a battlepass because making it singleplayer casual is better for games of this type.
